# React-Tic-Tac-Toe-Game
Project developed with React recreating the Tic Tae Toe game.

## How to install packages

## Walk to the app folder

```sh
$ cd app
```

### yarn

```sh
$ yarn
```

### npm

```sh
$ npm install
```

## How to run application

### yarn
```sh
$ yarn start
```

### npm

```sh
$ npm start
```
